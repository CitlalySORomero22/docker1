¿Que es Ubuntu?


Ubuntu es uno de los sistemas operativos basados en Linux más utilizado en la actualidad, algo que ha conseguido gracias a su rendimiento, a la posibilidad de descargarse gratis y tratarse de software libre que cualquiera puede modificar para mejorar su código, algo que ha propiciado la aparición de otros entornos gráficos como Kubuntu o Xubuntu entre muchos otros.

Docker empaqueta software en unidades estandarizadas llamadas contenedores que incluyen todo lo necesario para que el software se ejecute, incluidas bibliotecas, herramientas de sistema, código y tiempo de ejecución. Con Docker, puede implementar y ajustar la escala de aplicaciones rápidamente en cualquier entorno con la certeza de saber que su código se ejecutará.

Por qué usar Ubuntu

Hay muchas razones para usar Ubuntu Linux que lo convierten en una distribución de Linux digna. Además de ser gratuito y de código abierto, es altamente personalizable y tiene un Centro de software lleno de aplicaciones. Existen numerosas distribuciones de Linux diseñadas para satisfacer diferentes necesidades.
